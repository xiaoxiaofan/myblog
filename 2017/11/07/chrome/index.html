Chrome version

发表于 2017-11-07

Chrome 56 Beta

Date:2016.12.8

主要更新：

    1. 安全警告（对于http包含密码的请求）
    1. web蓝牙
    1. css position sticky的支持
  • 其它： Remote Playback API ，WebVR API，WebGL 2.0 API ，WebGL 2.0 API 等等。

Chrome 无版本

Date:2017.1.26

主要更新：

    1. 精简加载过程，更快的重载。

Chrome 无版本

Date:2017.1.30

主要更新：

    1. 渲染流水线的提高，提升性能。选择更优的渲染算法，更好利用系统硬件，load更加快，更平滑，更节能。

Chrome 无版本

Date:2017.2.2

主要更新：

    1. 深度集成PWA（Progressive Web Apps）到Android中。

Chrome 57 Beta

Date:2017.2.2

主要更新：

    1. CSS Grid Layout可以使用
    1. Improved Add to Home screen
    1. Media Session API
  • 其它： Fetch API Response ,PaymentMethodData等等。

Chrome 无版本

Date:2017.3.16

主要更新：

    1. Faster 3D rendering with WebGL 2.0

Chrome 58 Beta

Date:2017.3.21

主要更新：

    1. IndexedDB 2.0全面支持
    1. an improvement to iframe navigation（帧导航提升）
    1. immersive full screen for PWAs（沉浸式的全屏）
  • 其它： ControlsList API ，improved Add to Homescreen flow等等。

Chrome 无版本

Date:2017.4.11

主要更新：

    1. JavaScript performance

Chrome 59 Beta

Date:2017.5.2

主要更新：

    1. Headless Chromium
    1. Native notifications on macOS
    1. Service worker navigation preload
  • 其它：Streams API 等等。

Chrome 60 Beta

Date:2017.6.13

主要更新：

    1. Paint Timing API
    1. CSS font-display
    1. Credential Management API improvements

Chrome 61 Beta

Date:2017.8.15

主要更新：

    1. JavaScript modules声明依赖
    1. Payment Request API on desktop
    1. Web Share API
    1. WebUSB
  • 其它：Network Information API 等等。

Chrome 62 Beta

Date:2017.9.20

主要更新：

    1. Network Quality Estimator API

  • 其它：等等。
